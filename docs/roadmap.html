Wafer Polishing Data Manager is a desktop application for CMP (Chemical Mechanical Polishing) research using ARACA equipment. It automates raw sensor data processing, computes derived metrics (COF, forces, pressure), and provides interactive analysis — including PCA clustering, correlation exploration, and predictive modeling — with Excel & PowerPoint report generation.
