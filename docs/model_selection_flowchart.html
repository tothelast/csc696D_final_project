How many samples?
Under 200
Start simple: Ridge, Lasso, or conservative CatBoost
200–10,000
Gradient boosting (CatBoost, XGBoost, LightGBM) is usually best
10,000+
Deep learning becomes viable; gradient boosting still competitive
Mostly categories or numbers?
Mostly categories
CatBoost (native), LightGBM, or tree-based models. Avoid plain Ridge.
Mostly numbers
Ridge, Lasso, SVR, neural networks all work naturally
Mixed
Gradient boosting handles both well
Linear or non-linear relationships?
Linear
Ridge or Lasso. Simple, fast, exact solution.
Unknown or non-linear
Trees (they handle both). Use residuals from Ridge to check.
Complex patterns
Gradient boosting or neural networks
Do interactions matter?
Yes or maybe
Tree-based models find these automatically. Ridge can't without manual feature engineering.
No
Ridge/Lasso work great — simpler is better when interactions aren't present
Features > samples?
Yes (wide data)
Lasso or ElasticNet — they zero out irrelevant features. Or PCA to reduce dimensions first.
No (tall data)
No special handling needed. Most models work.
Missing data?
Yes
CatBoost, XGBoost, LightGBM handle it natively. Ridge/Lasso need you to fill in gaps first.
No
Any model works
Outliers present?
Yes
Tree-based models are robust. Ridge/Lasso get dragged by outliers — use Huber loss or remove outliers first.
No or removed
Any model works
Need to explain predictions?
Yes, to non-technical
Ridge (clear coefficients) or shallow tree (visual rules)
Somewhat
CatBoost + SHAP values (shows each feature's push/pull per prediction)
Not needed
Neural networks, deep ensembles — optimize for accuracy only
Our CMP project traced through this flowchart
76 samples (small) + mostly categorical + interactions likely + need interpretability → CatBoost with conservative settings as primary, Ridge as interpretable baseline.

This is why we landed on the recommendation we did. A different dataset — say 50,000 rows of all-numerical sensor data with no missing values — would trace a completely different path and land on a completely different model. The data decides.
